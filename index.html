    • Python is what I call a pre compiled language.

    In C++ and several other programming languages you need to compile your code before the computer can use it. Each computer OS handles machine code differently. So if you want to develop a program for both IOS and Windows you have to write 2 different programs to be able to run the program on either machine.

    Python and Java amongst other languages use an interperter to run the program which compiles it on the fly so it will run on the platform you want it to while only having to write one program.
