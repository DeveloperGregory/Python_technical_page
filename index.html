    • What is Python

  • Python was created in 1991 by Guido van Rossum. Rossum's goal was to create a programming language that emphasized code readability.

    The language is an interpreted, high-level and general-purpose programming language. Used for a myriad of applications.

  • Python's readability shows itself by using an object oriented approach allowing programmers to create clean and concise code.

    Python is currently on version 3.9.0.

  • Python was created in the late 80's as a followup to the ABC programming language.

    Python 2.0 was released in 2000 introducing features like list comprhension.

  • Python interupters are avaialbe for many operating systems including but not limited to Linux, iOS, and Windows.

    Python 3.0 was released in 2008 and Python 2 was officially retired in 2020.

  • Python is really popular for coding machine learning programs.

    For more on Python you can go to their website: python.org

    • Printing to the Screen

  • To do display text on the screen:

    print(what you want to print)

    The print command will display text like:

    print("Hello World")

  • If you notice the quote marks around the text it tells Python that everything in the quotes should considered a string.

    You can also use print to display information about variables which you will learn more about in the next section.

    variable = "Hello World"
    print(variable)

  • To concatanate a string with a +:

    word1 = "Hello"
    word2 = "World"
    print(word1 + ", " + word2 + "!")

    Will display:

    Hello, World!

  • Use the .format command to insert variables without concatanting

    print('{0} and {1}'.format('Python', 'Documentation'))

    Will produce:

    Python and Documentation

    List are ways to store many points of data into one variable for later retrieval

    Let's take the following example which will only use variables

    myName = "Jimmy"
    myAge = 31
    is_married = False

    Then any time we had to access the varibles and get information we would have to name all 3

    print(myName + " " + myAge + " " + is_married)

    Which quickly becomes tedious. A list is easier to create and access when needed.

    user_info = ["Jimmy",31, False]

    Tuples

    A way to store mutiple pieces of data in one variable that cannot be changed after the creation of the tuple.

    myTuple = (42, "Hello, World", True)

    The above tuple now has 3 diffirent pieces of data store in it. The parenthesis around the items tells Python that it is a tuple.

    You can then later access the data by accessing the variable and then which item you want that it is storing. Keep in mind Python like most programming languages begin counting at 0.

    print(myTuple(0))

    Will display

    42
