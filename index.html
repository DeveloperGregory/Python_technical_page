What is Python

Python is an all purpose, high level and interpreted language. It was created in the late 1980's by Guido van Rossum as followup to the ABC programming language. Guido Van Rossum's goal was to create a programming language that emphasized code readability. Version 1.0 was released in 1991. Nine years later in 2000 version 2.0 was released. In 2008 version 3.0 was released. In 2020 version 2.0 was officially retired.

Python is currently on version 3.9.0.

For more on Python you can go to their website: python.org

Printing to the Screen

To do display text on the screen:

print(what you want to print)
An example of this is:
print("Hello World")
Hello World
print() is a method which is explained further in this document. Some methods can be passed paramaters. The example above shows that "Hello World" is the parameter being passed to print() another parameter you can pass to print() is a variable which is described in more detail in the next section.
variable = "Hello World"
print(variable)
Hello World

Use the .format() method to insert variables inside your print() statement.

print('{0} and {1}'.format('Python', 'Documentation'))

Variables

Python uses three main data types for storing variables. String, Numeric, and Boolean.

  • Strings

    A string is text that is surrounded by quotes. string_example = "This is a string!"

    Strings store whatever you put inside the quotes except for when you use quotes inside the quotes you have to be careful on which quote you use to define the string and which quote is a part of the string. bad_string = "I am the "worst"" this will result in an error or the variable not being stored correctly. good_string = 'I am the "best"?' which will work with no problems.

    You can convert any non-string into a string by using str()

    not_string = 6 #No quotes around the 6 means Python stores it as a number not a string
    print(str(not_string))
    6
    This is nessecary when concatenating diffrent data types.

    Select different characters in a string by using [] notation. Keep in mind like most programming languages Python starts counting from zero which would be the first character in a string.

    my_string = "Howdy"
    print(my_string[4])
    t

    len() will return the length of your string. The count begins at 1 instead of zero like the [] notation.

    my_string = 'Count the length'
    print(len(my_string))

    Tuples

    A way to store mutiple pieces of data in one variable that cannot be changed after the creation of the tuple.
