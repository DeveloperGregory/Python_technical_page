    • What is Python

    Python is a interperted language. The code is passed through an interputer.

    In C++ and several other programming languages you need to compile your code before the computer can use it. Each computer OS handles machine code differently. So if you want to develop a program for both IOS and Windows you have to write 2 different programs to be able to run the program on either machine.

    Python and Java amongst other languages use an interperter to run the program which compiles it on the fly so it will run on the platform you want it to while only having to write one program.

    Printing to the Screen

    Displaying text on the screen is as simples as:

    print(what you want to print)

    The print command will print text like

    print("Hello World")

    If you notice the quote marks around the text it tells Python that everything in the quotes should considered a string. A string is another way of saying text.

    You can also use print to display information about variables which you will learn more about in the next section.

    variable = "Hello World"
    print(variable)

    Variables

    A variable is a way to store information for future use.

    To create a variable create a name which has signficance to the data you are storing.

    myVariable = "An example of Variables"

    Variables store different data types:

    String - A collection of text, numbers, punctuation, and everything that is not one of the other data types.

    this_is_a_string = "This is a string."

    Number - Every digit that mathmatics can be applied to.

    this_is_a_number = 42

    Boolean - True or False

    this_is_a_boolean = True

    Tuples

    A way to store mutiple pieces of data in one variable that cannot be changed after the creation of the tuple.

    myTuple = (42, "Hello, World", True)

    The above tuple now has 3 diffirent pieces of data store in it. The parenthesis around the items tells Python that it is a tuple.

    You can then later access the data by accessing the variable and then which item you want that it is storing. Keep in mind Python like most programming languages begin counting at 0.

    print(myTuple(0))

    Will display

    42
