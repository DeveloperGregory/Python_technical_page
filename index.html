What is Python

Python is an all purpose, high level and interpreted language. It was created in the late 1980's by Guido van Rossum as followup to the ABC programming language. Guido Van Rossum's goal was to create a programming language that emphasized code readability. Version 1.0 was released in 1991. Nine years later in 2000 version 2.0 was released. In 2008 version 3.0 was released. In 2020 version 2.0 was officially retired.

Python is currently on version 3.9.0.

This is meant to be a starter technical page for Python and leaves a lot out. For more on Python you can go to their website: python.org

Printing to the Screen

To do display text on the screen:

print(what you want to print)
An example of this is:
print("Hello World")
Hello World
print() is a method and functions will be explained further in this document. Some methods can be passed paramaters. The example above shows that "Hello World" is the parameter being passed to print() another parameter you can pass to print() is a variable which is described in more detail in the next section.
variable = "Hello World"
print(variable)
Hello World

Use the .format() method to insert variables inside your print() statement.

print('{0} and {1}'.format('Python', 'Documentation'))

Variables

Python uses three main data types for storing variables. String, Numeric, and Boolean.

Strings

A string is text that is surrounded by quotes. string_example = "This is a string!"

Strings store whatever you put inside the quotes except for when you use quotes inside the quotes you have to be careful on which quote you use to define the string and which quote is a part of the string. bad_string = "I am the "worst"" this will result in an error or the variable not being stored correctly. good_string = 'I am the "best"?' which will work with no problems.

You can convert any non-string into a string by using str()

not_string = 6 #No quotes around the 6 means Python stores it as a number not a string
print(str(not_string))
6
This is nessecary when concatenating diffrent data types.

Select different characters in a string by using [] notation. Keep in mind like most programming languages Python starts counting from zero which would be the first character in a string.

my_string = "Howdy"
print(my_string[4])
y

len() will return the length of your string. The count begins at 1 instead of zero like the [] notation.

my_string = 'Count the length'
print(len(my_string))
16

There are several other methods for strings.

  • .upper() will return the string but with all the letters captalized.
  • .lower() will return the string but with all the letters in lowercase.
  • .title() will return the string but in Title Case.
  • .join() will connect the parameter to the string.
  • .isdigit() will return true if all the character in the string are digits.

    List

    List are like arrays in other programming languages. They are mutable and have no expansion limit. To create a list you use [] notation seperating each entry out with a comma. To access list items use [] notation. List like strings start counting at 0.

    my_lst = ["Johnny", 5, "is alive!"]
    print(my_lst[1])
    5
    You can also use [-1] to access the last element in the list.
    my_lst = ["Johnny", 5, "is alive!"]
    print(my_lst[-1])
    is alive!

    There are several methods that can be used on lists. Here is a few.

  • .append() will add items to the end of a list. my_lst.append(new_item)
  • .reverse() will return a list of the items but in reverse order.
  • .index() will return the index of the specified item.

    • List and strings are handled similiary thus you can use some of the same methods on list that you can use on a string. Like len() which will return how many items you have in that list. .join() will also add items to the list like it does with strings.

    Tuples

    Tuples are like list but are inmutable after creation. You can create a tuple by using () notation.

    my_tuple = ("No", "Disassemble", "Number", 5)
    print(my_tuple)
    ("No", "Disassemble", "Number", 5)
    You access items in a tuple the same as you would a list except you use () notation.
    my_tuple = ("No", "Disassemble", "Number", 5)
    print(my_tuple(2))
    Number

    If statement

    If statements return True or False depending on the outcome of the variables being compared. The basic structure of an if statement is as follows:

    my_num = 42
    if(my_num == 42):
    ....print("The Answer") #The dots before print are to denote the 4 spaces tabbed in. Python uses whitespace to signify that this line is part of the if statement
    The Answer

    There are several operators for if statements can use:

  • <= Less than or equal to if(6 <= 4) would evaluate to False
  • >= Greater than or equal to if(7 >= 2) would evaluate to True
  • > Greater than if(42 > 42) would evaluate to False
  • < Less than if(15 < 17) would evaluate to True
  • != Not equal to if("Johnny" != "Daniel") would evaluate to True

    • You can combine elif: and else: statements to extend the functionality of an if statement.

    ....print('That is a DC comic')
    else: #will catch anything that the other ifs and elifs did not catch
    ...print('That is a comic I don't know.'')
    That is a DC comic.
